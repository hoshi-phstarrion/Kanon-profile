かのんのホームページ成長記録

2025年4月10日

ホームページを開設しました！
GitHub Pagesを使って、自己紹介サイトの第一歩を踏み出しました。
最初は空っぽのページからのスタートだったけど、ここから少しずつ育てていく予定です。

2025年4月14日

  1. メニューバーの改善
    ナチュラルな角丸ボックス型ナビゲーションに変更。Font Awesomeで各リンクにアイコンを追加。区切り線を小さな丸（fa-circle）でやさしく表現。コンパクトな配置でスマート＆可愛いを両立！

  2. ことばの部屋の追加
    かのんオリジナルの詩を掲載するセクションを新設。柔らかい色合いのボックスで詩の雰囲気を表現。セクションタイトルにも下線装飾を追加し、統一感アップ。

  3. お問い合わせフォームの導入
    Formspreeを使ったメール送信フォームを導入。名前・メールアドレス・メッセージを簡単に送信可能。HTMLだけで実装できるため、GitHub Pagesでもすぐに使える。

  4. カラーテーマ＆スタイルの調整
    全体にやさしいパステルカラーを採用（背景やアクセント色）。各セクションタイトル（h2）に下線を追加して視認性UP。トップ画像に角丸＋ふんわり影を追加し、柔らかさを演出。

  5. GitHubプロフィールリンクの追加
    サイトのフッターにGitHubプロフィールリンクを追加。アイコン付きで、かのんの活動がすぐに見られるように。

2025年4月14日（追加分）

  1. 鑑賞記録ノートの開始
    新たに「みたもの」ページを追加し、「鑑賞記録ノート」というタイトルに変更。
    初めての記録として『ちひろさん』の鑑賞内容を丁寧にまとめました。

2025年4月15日

  1. トップページのタイトルと説明の改善
    サイトタイトルを「斎藤かのんの小さなまほうの箱」に変更し、
    検索結果やSNS共有時に伝わるように meta description も追加しました。

  2. ランダムに表示される詩の導入
    「ことばの部屋」に JavaScript を使ってランダム表示機能を追加。
    見るたびにちがうやさしい言葉が出迎えてくれるように。

  3. スムーススクロールの導入
    サイト内リンクがふんわりスクロール移動するようになりました。
    おちついた印象で、やさしい雰囲気を強化。

  4. フッターに“かのんからのひとこと”を追加
    読み終わったあとに、そっと心を包むような一言を追加。

  5. お問い合わせフォームに補足を追加
    「お名前（ニックネームでもOK）」という表記に変更して、
    もっと気軽に連絡できるように改善しました。
